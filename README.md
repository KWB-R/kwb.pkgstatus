# kwb.pkgstatus

R package for checking KWB package status (e.g. generating https://kwb-r.github.io/status)

## Installation

```r
#install.packages("devtools", repos = "https://cloud.r-project.org")
devtools::install_github("KWB-R/kwb.pkgstatus")
```
